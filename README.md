## ✨ About

A modern, responsive personal portfolio website showcasing my journey as a Software Developer, AI & ML Enthusiast, and Data Analyst. Built with React + TypeScript.

## 🚀 Features

- **Modern UI** — Clean design with gradient themes, smooth animations, and glassmorphism effects
- **Responsive** — Mobile-friendly with hamburger menu navigation
- **Projects Showcase** — Interactive project cards with tech tags
- **Featured Projects** — 
  - LLM Research Journey @ Ericsson (timeline + stats visualization)
  - Game UX Analytics @ Ruse University (data dashboard with animated charts)
- **Embedded Videos** — Google Drive video integration for project demos
- **My Life Story** — Personal storytelling section

## 🛠️ Tech Stack

- React 18
- TypeScript
- React Router (HashRouter)
- CSS3 (Flexbox, Grid, Animations)
- GitHub Pages
